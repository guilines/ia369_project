Projeto

Unicamp - IA369 W


Esse projeto foi desenvolvido para a disciplina de visualização de dados (IA369W), lecionada pela professora Paula Dornhofer Paro Costa para pós-graduação na Faculdade de Engenharia Elétrica (FEEC) da Universidade Estadual de Campinas durante o primeiro semestre de 2017.

O projeto tem como objetivo conduzir um estudo prático sobre percepção humana aplicada à visualização de dados, possibilitando uma comparação dos resultados com referências estudadas na disciplina.

Os resultados apresentados são preliminares, visto que uma pesquisa maior e mais controlada seria necessária para resultados definitivos, o que foge do escopo do projeto. Este projeto pode ser encontrado no github: ia369_project
O relatório completo está disponível em: Report
