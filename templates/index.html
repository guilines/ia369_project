A ferramenta de visualização proposta tem como objetivo permitir que o usuário navegue por uma série de dados distintos que usualmente não estão facilmente disponíveis em conjunto. Um exemplo disso é a possibilidade de visualizar dados de evolução do grau de urbanização em conjunto com dados de mortalidade infantil entre 1990 e 2012, permitindo que o usuário adquira bastante informação pela fácil exploração dos indicadores.

Neste projeto utilizamos o conjunto de dados chamado “Indicadores e dados Básicos - IDB - 2012”, que reúne informações demográficas, econômicas e sobre a saúde brasileira entre os anos de 1990 e 2012.

Fonte dos dados: http://tabnet.datasus.gov.br/cgi/idb2012/matriz.htm#demog
